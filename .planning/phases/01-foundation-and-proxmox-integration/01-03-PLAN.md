<!-- API contracts from Plans 01 and 02 that the UI calls -->

Settings API (from Plan 01):
- GET /api/settings -> Returns all settings grouped by prefix
- PUT /api/settings -> Body: Record<string, string>, Returns: { saved: boolean, validation?: { valid: boolean, error?: string, nodeName?: string } }
- POST /api/proxmox/validate -> Returns: { valid: boolean, error?: string, nodeName?: string }

Container API (from Plan 02):
- GET /api/proxmox/containers -> Returns ContainerInfo[]
- POST /api/proxmox/containers -> Body: { vmid, hostname, ostemplate, memory?, cores?, cameraName?, cameraIp?, cameraType? }, Returns: { status: 'created'|'updated', vmid }
- DELETE /api/proxmox/containers/[vmid] -> Returns { success: boolean, vmid }
- POST /api/proxmox/containers/[vmid]/start -> Returns { success: boolean, vmid, action: 'started' }
- POST /api/proxmox/containers/[vmid]/stop -> Returns { success: boolean, vmid, action: 'stopped' }
- POST /api/proxmox/containers/[vmid]/restart -> Returns { success: boolean, vmid, action: 'restarted' }

Server-side data loading:
```typescript
// src/routes/+layout.server.ts -- provides settings status to all pages
import { getSettings } from '$lib/server/services/settings';
export async function load() {
  const proxmox = await getSettings('proxmox_');
  const unifi = await getSettings('unifi_');
  return {
    configured: {
      proxmox: !!proxmox.proxmox_host && !!proxmox.proxmox_token_id,
      unifi: !!unifi.unifi_host
    }
  };
}

// src/routes/kameras/+page.server.ts -- provides container list
import { listContainers } from '$lib/server/services/proxmox';
export async function load() {
  try {
    const containers = await listContainers();
    return { containers };
  } catch {
    return { containers: [], error: 'Could not load containers' };
  }
}
```

       ```typescript
       import { getSettings } from '$lib/server/services/settings';
       import type { PageServerLoad } from './$types';

       export const load: PageServerLoad = async () => {
         const proxmox = await getSettings('proxmox_');
         const unifi = await getSettings('unifi_');
         const allSettings = await getSettings('credential_');
         // Parse credential settings into structured list
         const credentialNames = new Set<string>();
         for (const key of Object.keys(allSettings)) {
           const match = key.match(/^credential_(.+?)_(username|password|ip)$/);
           if (match) credentialNames.add(match[1]);
         }
         const credentials = [...credentialNames].map(name => ({
           name,
           username: allSettings[`credential_${name}_username`] || '',
           cameraIp: allSettings[`credential_${name}_ip`] || ''
         }));
         return { proxmox, unifi, credentials };
       };
       ```
