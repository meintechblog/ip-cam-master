        * Generates go2rtc YAML for a Bambu Lab A1 printer.

        * port 6000. We spawn the ingestion Node script via go2rtc's exec: pipe

       export function generateGo2rtcConfigBambuA1(params: {
           streamName: string;
           printerIp: string;
           accessCode: string;
           rtspAuth?: RtspAuth;
