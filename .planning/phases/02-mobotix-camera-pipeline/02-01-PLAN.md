    **1. Create onboarding service** (`src/lib/server/services/onboarding.ts`):

    Import from ssh.ts, go2rtc.ts, proxmox.ts, settings.ts, db/client.ts, db/schema.ts, types.ts.

    - `testMobotixConnection(ip: string, username: string, password: string)`:
      Attempt RTSP probe. Connect to Proxmox via SSH, then run a quick ffprobe on the host itself (not in a container):
      `ssh.execCommand(\`ffprobe -v quiet -print_format json -show_streams -timeout 5000000 rtsp://${username}:${password}@${ip}:554/stream0/mobotix.mjpeg 2>&1 || echo PROBE_FAILED\`)`
      Parse stdout for video stream info (width, height, r_frame_rate). If PROBE_FAILED, try alternate path `/mobotix.mjpeg`. If both fail, try simple TCP connect to port 554 as fallback (just confirm camera is reachable).
      Return `{ success: boolean, resolution?: string, fps?: number, streamPath?: string, error?: string }`.

    - `saveCameraRecord(params: { name, ip, username, password, cameraType, streamPath, width, height, fps, bitrate, vmid, streamName })`:
      Insert into cameras table. The `streamName` defaults to `cam-${vmid}`. Encrypt password using the crypto module (import `encrypt` from `./crypto`).
      Return the inserted camera id.

    - `createCameraContainer(cameraId: number)`:
      Read camera record from DB by id. Read proxmox settings for vmid_start. The vmid should already be assigned in the camera record.
      Call `proxmox.createContainer({ vmid: camera.vmid, hostname: \`cam-${camera.vmid}\`, ostemplate: 'local:vztmpl/debian-12-standard_12.7-1_amd64.tar.zst', cameraName: camera.name, cameraIp: camera.ip, cameraType: camera.cameraType })`.
      Call `proxmox.startContainer(camera.vmid)`.
      Connect SSH, call `waitForContainerReady(ssh, camera.vmid)`.
      Get container IP: `executeOnContainer(ssh, camera.vmid, 'hostname -I')` -- trim stdout, take first IP.
      Update camera record: set containerIp, status = 'container_created'.
      Dispose SSH.
      Return `{ vmid: camera.vmid, containerIp }`.

    - `configureGo2rtc(cameraId: number)`:
      Read camera record. Connect SSH.
      Run install commands from `getInstallCommands()` via `executeOnContainer`.
      Generate YAML config via `generateGo2rtcConfig({ streamName: camera.streamName, cameraIp: camera.ip, username: camera.username, password: decrypt(camera.password), width: camera.width, height: camera.height, fps: camera.fps, bitrate: camera.bitrate })`. Note: decrypt password before passing.
      Push config via `pushFileToContainer(ssh, vmid, yamlContent, '/etc/go2rtc/go2rtc.yaml')`.
      Generate systemd unit, push to `/etc/systemd/system/go2rtc.service`.
      Run `executeOnContainer(ssh, vmid, 'systemctl daemon-reload && systemctl enable go2rtc && systemctl restart go2rtc')`.
      Update camera status = 'configured'.
      Dispose SSH.

    - `verifyStream(cameraId: number)`:
      Read camera record. Call `checkStreamHealth(camera.containerIp!, camera.streamName)`.
      If active, set rtspUrl = `rtsp://${camera.containerIp}:8554/${camera.streamName}`, status = 'verified'.
      Return `{ success: boolean, rtspUrl?: string, streamInfo: StreamInfo }`.

    - `getNextVmid()`: Read `proxmox_vmid_start` from settings (default 200). Query cameras table for max vmid. Return max(vmid_start, max_vmid + 1).
